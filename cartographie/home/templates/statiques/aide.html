Étudiants et futurs étudiants, enseignants, chercheurs, responsables d'établissement

Vous souhaitez consulter la liste des formations francophones -
un court film vous offre une démonstration de l'utilisation de ce site : voir la démo.

Deux options s'offrent à vous pour trouver une formation :

Carte

Pour trouver des formations dispensées dans un pays en particulier, cliquer sur le pays ou territoire concerné.

Vous arriverez directement sur la page de résultats de recherche avec toutes les formations offertes pour le pays sélectionné.
Vous pourrez ensuite affiner votre recherche.

Rechercher

La zone de recherche présente toutes les formations disponibles sur ce site.
Pour cibler les formations qui vous intéressent, trois options intuitives s'offrent à vous.

1. Recherche textuelle

Le moteur de recherche textuel actuel permet uniquement des recherches simples.
Saisir une chaîne de caractères exacte, la recherche de cette chaîne se fera dans :

  • l'intitulé de la formation
  • les disciplines associées à la formation
  • le niveau de diplôme de la formation
  • le nom de l'établissement qui dispense la formation
  • le nom du pays où est dispensée la formation
  • le nom de la région* où est dispensée la formation

Nous travaillons actuellement pour mettre à votre disposition un moteur de recherche textuel plus puissant.

2. Recherche avancée

Pour affiner votre recherche, choisir parmi les filtres suivants :

  • Région* : la région où est dispensée la formation
  • Pays : le pays où est dispensée la formation
  • Établissement : l'établissement membre de l'AUF qui dispense la formation
3. Naviguer

Vous pouvez également naviguer dans les disciplines principales associées aux formations.

Les disciplines sont organisées hiérarchiquement.
Pour chacune d'elles, le nombre de formations disponibles est fourni entre parenthèses
(incluant les formations associées à ses sous-disciplines).

Cliquer sur une discipline pour :

  • Limiter les résultats aux formations associées à cette discipline (ou ses sous-disciplines)
  • Afficher ses sous-disciplines pour pouvoir continuer la navigation et cibler davantage

Le référentiel adopté pour cela est le classement des disciplines utilisé par le programme européen Erasmus.

Connexion

Actuellement, aucune connexion n'est nécessaire pour consulter le site.

Représentant d'un établissement membre de l'AUF

Vous travaillez pour un établissement membre de l'AUF et vous souhaitez
ajouter, modifier ou supprimer des formations pour votre établissement :
Merci de contacter le bureau régional de l'AUF de votre région.

Employé AUF

Vous êtes un employé de l'AUF et vous avez besoin d'aide, contactez l'équipe du projet Cartographie.

* Les régions correspondent aux zones couvertes par les dix bureaux régionaux de l'AUF, voir la liste sur le site institutionnel de l'AUF.
