Étudiants, enseignants, chercheurs... ou grand public

Un court film vous offre une démonstration de l'utilisation de ce site : voir la démo.
Deux options s'offrent à vous pour trouver une formation francophone.

Carte

Pour trouver des formations dispensées dans un pays en particulier, simplement cliquer sur le pays ou territoire concerné.

Vous arriverez directement sur la page de résultats de recherche avec toutes les formations offertes pour le pays sélectionné.
Vous pourrez ensuite raffiner votre recherche, au besoin.

Rechercher

La zone de recherche vous offre toutes les formations disponibles sur ce site.
Pour cibler les formations qui vous intéressent, trois options intuitives s'offrent à vous.

1. Recherche textuelle

Le moteur de recherche textuel actuel ne permet seulement qu'une recherche simple.
Saisir une chaîne de caractères exacte, la recherche de cette chaîne se fera dans :

  • le libellé de la formation
  • les disciplines associées à la formation
  • le niveau de diplôme de la formation
  • le nom de l'établissement membre de l'AUF dispensant la formation
  • le nom du pays où est dispensée la formation
  • le nom de la région AUF où est dispensée la formation

L'AUF travaille à mettre à votre disposition un moteur de recherche textuel plus puissant.

2. Recherche avancée

Pour affiner votre recherche, choisir les filtres désirés parmi :

  • Région : la région de l'AUF où est dispensée la formation
  • Pays : le pays où est dispensée la formation
  • Établissement : l'établissement membre de l'AUF dispensant la formation
3. Naviguer

Vous pouvez aussi simplement naviguer dans les disciplines principales associées aux formations.

Les disciplines sont organisées hiérarchiquement.
Pour chacune d'elle, le nombre de formations disponibles est fourni entre parenthèses
(incluant les formations associées à ses sous-disciplines).

Cliquer sur une discipline pour :

  • Limiter les résultats aux formations associées à cette discipline (ou ses sous-disciplines)
  • Afficher ses sous-disciplines pour pouvoir continuer la navigation et cibler davantage

Pour information, le référentiel utilisé pour les disciplines est le référentiel utilisé dans le cadre du projet européen Erasmus.

Connexion

Actuellement, aucune connexion n'est disponible pour les étudiants, enseignants, chercheurs ou le grand public.
Seuls les employés de l'AUF peuvent se connecter afin de pouvoir gérer les formations offertes.

Établissement membre de l'AUF

Si vous travaillez pour un établissement membre de l'AUF et que vous souhaitez
ajouter, modifier ou supprimer des formations pour votre établissement :
merci de contactez le bureau régional de l'AUF qui dessert votre région.

Employés AUF

Si vous êtes un employé de l'AUF et avez besoin d'aide, contactez l'équipe projet cartographie.
